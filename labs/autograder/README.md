# Overview

Tool for automatically generating Gradescope autograder from a specification file. Most labs are built using this tool. See `lab-11` for an example.

To build `lab-11`, run
```
python3 build_autograder.py ../lab-11/
```

If the autograder is automatically generated, do not modify it. Instead, modify the handout or specification file, then rebuild the autograder.
